## 📊 Portfolio Highlights

### Project 1 — Placement Management Analytics System
- **Problem:** Manual Excel-based placement tracking with no prediction capability
- **Solution:** 4-table SQL database + Python EDA + Random Forest (86% accuracy) + Power BI dashboard
- **Impact:** Early at-risk student identification 12 months ahead; 30% reporting time saved

### Project 2 — Social Media Sentiment & Brand Intelligence
- **Problem:** No systematic way to detect brand reputation shifts before they become crises
- **Solution:** 30,000 post NLP pipeline with VADER + TextBlob + LDA + Power BI crisis detection
- **Impact:** Z-score spike detection enables 48-72hr early crisis warning

### Project 3 — Retail Sales Forecasting & Insight Engine
- **Problem:** Manual reporting consuming 70% of analyst time; no forecast capability
- **Solution:** Prophet forecasting (MAPE 1.75%) + RFM segmentation + cohort analysis + Power BI
- **Impact:** 70% reporting reduction; 7 customer segments for targeted marketing

### Project 4 — Redrob Candidate Ranking Engine
- **Problem:** Recruiting top-tier AI Engineers is plagued by resume padding and volume overload
- **Solution:** Feature engineering + XGBoost + 3-Tier Honeypot Detection
- **Impact:** Filters 100,000+ synthetic profiles to maintain a top-100 leaderboard without global sorts
